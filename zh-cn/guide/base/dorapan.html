    # 用户空间 SDK - Dorapan

    Who is Dorapan

    因为 Pandora.js 可能是全局安装的，在用户项目中无法直接 require('pandora')，存在各种问题。推荐统一使用 require('dorapan') 来引用 Pandora.js 导出的运行时上下文信息、基础类等。

    # 在用户目录下安装 dorapan

    tnpm i dorapan --save

    # require('dorapan') 等价于 require('pandora')

    console.log(require('dorapan') === require('pandora')); // true
