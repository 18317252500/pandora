# 日志自动清理

Pandora.js 暂不提供日志清理功能，但 Pandora.js 已经将日志按天切割（或尺寸）。可以针对 ~/logs 配置一个简单 crontab，如类似教程。

# 日志等级

  1. ALL - 全部等级
  2. DEBUG - DEBUG 及其以下
  3. INFO - INFO 及其以下
  4. WARN - WARN 及其以下
  5. ERROR - ERROR 及其以下
  6. NONE - 全部不输出
