## Installation

Please use tag `beta` before we release!

```bash
$ npm install pandora@beta --save
```

Node.js >= 6.0.0 required.

## Features

- ✔︎ Built-in application model
- ✔︎ Built-in process management
- ✔︎ Dev Ops customization
- ✔︎ Support lots of web framework
